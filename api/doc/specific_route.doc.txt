Ce document décrit les routes spécifique à certaines
ressources de l'API

<----- SUMMARY ---------------------------------------------->
1. USER AUTHENTICATION
2. USER CHANGER DE MDP
3. TOPIC GET ALL MSG
4. TOPIC GET PAGE MSG
5. TOPIC GET MSG COUNT
6. TOPIC GET FRESH
<------------------------------------------------------------>


1.
<------------------------------------------------------------>
<--- USER AUTHENTICATION ---------------------------------->
<------------------------------------------------------------>
Permet d'authentifier un utilisateur grâce au couple
username/password

--- PATH -----------------------------------------------------
/user/auth

--- REQUEST METHOD -------------------------------------------
post

--- PARAMETERS -----------------------------------------------
| NAME        | METHOD          | DESC                       |
| username    | post            | le login de l'utilisateur  |
| password    | post            | le mdp en clair            |
--------------------------------------------------------------

--- RETURN VALUE ---------------------------------------------
| CASE                              | STATUS | DATA TYPE     |
| success                           | 200    | none          |
| incorrect username                | 403    | none          |
| incorrect password                | 403    | none          |
--------------------------------------------------------------


2.
<------------------------------------------------------------>
<--- USER CHANGER UN MOT DE PASSE --------------------------->
<------------------------------------------------------------>
Change le mot de passe d'un utilisateur
<!> ne demande pas l'ancien mdp (à implémenter) <!>

--- PATH -----------------------------------------------------
/user/password

--- REQUEST METHOD -------------------------------------------
post

--- PARAMETERS -----------------------------------------------
| NAME        | METHOD          | DESC                       |
| id          | post            | id de l'user               |
| password    | post            | le nouveau mdp             |
--------------------------------------------------------------

--- RETURN VALUE ---------------------------------------------
| CASE                              | STATUS | DATA TYPE     |
| success                           | 200    | none          |
--------------------------------------------------------------


3.
<------------------------------------------------------------>
<--- TOPIC GET ALL MSG -------------------------------------->
<------------------------------------------------------------>
Renvoi tout les messages du topic dont l'id est passé en
paramètre

--- PATH -----------------------------------------------------
/topic/:id/msg

--- REQUEST METHOD -------------------------------------------
get

--- PARAMETERS -----------------------------------------------
| NAME        | METHOD          | DESC                       |
| id          | get             | l'id du topic              |
--------------------------------------------------------------

--- RETURN VALUE ---------------------------------------------
| CASE                              | STATUS | DATA TYPE     |
| success                           | 200    | json          |
| id not found                      | 404    | none          |
--------------------------------------------------------------


4.
<------------------------------------------------------------>
<--- TOPIC GET PAGE MSG ------------------------------------->
<------------------------------------------------------------>
Retourne les messages de la pages précisée en paramètre,
25 messages par page

--- PATH -----------------------------------------------------
/topic/:id/:page

--- REQUEST METHOD -------------------------------------------
get

--- PARAMETERS -----------------------------------------------
| NAME        | METHOD          | DESC                       |
| id          | get             | l'id du topic              |
| pages       | get             | le numéro de la pages      |
--------------------------------------------------------------

--- RETURN VALUE ---------------------------------------------
| CASE                              | STATUS | DATA TYPE     |
| success                           | 200    | json          |
| no message at this index          | 404    | none          |
--------------------------------------------------------------


5.
<------------------------------------------------------------>
<--- TOPIC GET MSG COUNT ------------------------------------>
<------------------------------------------------------------>
Récupère le total de message posté dans un topic

--- PATH -----------------------------------------------------
/topic/:id/msgcount

--- REQUEST METHOD -------------------------------------------
get

--- PARAMETERS -----------------------------------------------
| NAME        | METHOD          | DESC                       |
| id          | get             | l'id du topic              |
--------------------------------------------------------------

--- RETURN VALUE ---------------------------------------------
| CASE                              | STATUS | DATA TYPE     |
| success                           | 200    | json          |
--------------------------------------------------------------


6.
<------------------------------------------------------------>
<--- TOPIC GET FRESH ---------------------------------------->
<------------------------------------------------------------>
récupère les topics par ordre du plus récemment actif
une page = 25 topics

--- PATH -----------------------------------------------------
/topic/fresh/:page  

--- REQUEST METHOD -------------------------------------------
get

--- PARAMETERS -----------------------------------------------
| NAME        | METHOD          | DESC                       |
|             |                 |                            |
--------------------------------------------------------------

--- RETURN VALUE ---------------------------------------------
| CASE                              | STATUS | DATA TYPE     |
| success                           | 200    | json          |
--------------------------------------------------------------
