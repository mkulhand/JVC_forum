// API DOCUMENTATION : GENERIC ROUTE MAP 					//
//////////////////////////////////////////////////////////////

Ce document décrit les routes permettant d'effectuer les
actions de bases de l'API

<----- SUMMARY ---------------------------------------------->
1. GET BY ID
2. AJOUTER UNE RESSOURCE
3. SUPPRIMER UNE RESSOURCE
<------------------------------------------------------------>


1.
<------------------------------------------------------------>
<--- GET BY ID ---------------------------------------------->
<------------------------------------------------------------>
Récupère la ressource correspondante a l'id passée en
paramètre

--- PATH -----------------------------------------------------
/<ressource_type>/:id

--- REQUEST METHOD -------------------------------------------
get

--- PARAMETERS -----------------------------------------------
| NAME        | METHOD          | DESC                       |
| id          | get             | id de la ressource         |
--------------------------------------------------------------

--- RETURN VALUE ---------------------------------------------
| CASE                              | STATUS | DATA TYPE     |
| success                           | 200    | json          |
| id not found                      | 404    | none          |
--------------------------------------------------------------


2.
<------------------------------------------------------------>
<--- AJOUTER UNE RESSOURCE ---------------------------------->
<------------------------------------------------------------>
Ajoute un ressource en BDD

--- PATH -----------------------------------------------------
/<ressource_type>/

--- REQUEST METHOD -------------------------------------------
put

--- PARAMETERS -----------------------------------------------
<-- user ---------------------------------------------------->
| NAME              | METHOD             | DESC              |
| username          | post               |                   |
| password          | post               |                   |
--------------------------------------------------------------

<-- topic --------------------------------------------------->
| NAME              | METHOD             | DESC              |
| title             | post               | titre du topic    |
| author_id         | post               | id de l'user      |
| msg               | post               | le text du 1er msg|
--------------------------------------------------------------

<-- msg ----------------------------------------------------->
| NAME              | METHOD             | DESC              |
| author_id         | post               | id de l'user      |
| topic_id          | post               | id du topic       |
| text              | post               | texte du msg      |
--------------------------------------------------------------

--- RETURN VALUE ---------------------------------------------
| CASE                              | STATUS | DATA TYPE     |
| success                           | 201    | none          |
--------------------------------------------------------------


3.
<------------------------------------------------------------>
<--- SUPPRIMER UNE RESSOURCE -------------------------------->
<------------------------------------------------------------>
Supprime une ressource

--- PATH -----------------------------------------------------
/<ressource_type>/:id

--- REQUEST METHOD -------------------------------------------
delete

--- PARAMETERS -----------------------------------------------
| NAME        | METHOD          | DESC                       |
| id          | get             | id de la ressource         |
--------------------------------------------------------------

--- RETURN VALUE ---------------------------------------------
| CASE                              | STATUS | DATA TYPE     |
| success                           | 200    | none          |
| id not found                      | 404    | none          |
--------------------------------------------------------------
